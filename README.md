# Курс Backend на Python. Весна '26

## Пререквизиты курса

### Обязательные

- Базовые знания Python (переменные, типы данных, функции, классы)
- Установленный Python 3.8+, IDE (рекомендуется PyCharm)

### Рекомендуемые

- Базовые знания работы с командной строкой
- Базовые знания работы с Git и GitHub
- Базовые знания HTML и HTTP

## Цель курса

Изучить написание Backend-сервисов на Python с нуля до уровня Junior. Научиться проектировать архитектуру приложений,
работать с базами данных, создавать и использовать API, писать тесты, деплоить сервисы, а также интегрировать LLM и разрабатывать боты для Telegram.

## Что вы узнаете на курсе (план по занятиям)

### Модуль 1. Введение в Backend-разработку на Python

- Что такое Backend и как используется Python для сервера? Объектно-ориентированное программирование (ООП) в Python
- Python для Backend: установка, настройка окружения, основные библиотеки

### Модуль 2. Git и разработка архитектуры приложений 
- Основы Git: создание репозитория, коммиты, ветки, слияния, командная разработка
- Проектирование архитектуры приложений: MVC, REST, микросервисы

### Модуль 3. Web-фреймворки и создание API
- HTTP-протокол: основы, методы, статусы, заголовки, отличия версий
- FastAPI: создание первого API, обработка запросов и ответов, pydantic, swagger

### Модуль 4. Работа с базами данных и асинхронностью
- Введение в базы данных: реляционные и нереляционные, SQL и NoSQL, ACID
- orm-библиотеки: SQLAlchemy, работа с PostgreSQL
- Асинхронное программирование: event loop, asyncio, aiohttp, асинхронные запросы

### Модуль 5. Тестирование
- Введение в тестирование: юнит-тесты, интеграционные тесты, pytest, mocks
- Безопасность веб-приложений: аутентификация, авторизация, защита от атак, шифрование данных

### Модуль 6. Docker и CI/CD
- Деплой приложений: Docker, CI/CD, GitHub Actions, environment variables
- Мониторинг и логирование: инструменты, настройка, анализ логов

### Модуль 7. Работа с LLM и разработка ботов для Telegram
- Введение в LLM: что это такое, основные модели, использование API
- Промпт-инжиниринг: создание эффективных промптов, оптимизация запросов
- Разработка ботов для Telegram: aiogram, создание своего бота

### Модуль 8. Бонусный модуль: создание нейросетей на Python
- Введение в нейросети: основы, архитектуры, библиотеки (TensorFlow, PyTorch)
- Создание и обучение простой нейросети на Python
- Применение нейросетей в реальных задачах

## Как будет проходить обучение
- Занятия будут проходить онлайн в формате лекций и практических занятий. **Не бойтесь задавать вопросы по ходу изучения материала!**
- Каждое занятие будет сопровождаться домашними заданиями для закрепления материала.
- На каждом занятии в начале будет краткий тест по пройденному материалу.
- По ходу курса будут несколько небольших проектов для практики.
- В конце курса будет проектная работа, где вы сможете применить все полученные знания на практике.
