# Python-backend-spring-26
Курс Backend-разработки на Python. Весна 2026

## Что вы узнаете на курсе

- Что такое бекэнд-разработка? Python, примитивы и основы разработки на нём
- Git, версионирование. Зачем это нужно?
- ООП: основные принципы - наследование, полиморфизм, инкапсуляция
- Архитектура приложений: как разрабатывать своё приложение, имея только идею
- Базы данных: что это такое, зачем это нужно. Реляционные и нереляционные БД. Основы PostgreSQL
- API: HTTP/0.9, HTTP/1.0, HTTP/2.0, чем они отличаются. Как работать с чужим API и разработать свой?
- Асинхронность: принципы асинхронности, асинхронные запросы
- Работа с LLM: как писать промпты и зачем это нужно?
- Боты Телеграм: aiogram
